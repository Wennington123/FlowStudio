# FlowStudio v1.1 🚀

**FlowStudio** é uma ferramenta desktop de alta precisão desenvolvida para automatizar a construção de fluxogramas de revisão sistemática, garantindo total conformidade visual e metodológica com a **Diretriz PRISMA 2020**.

---

## 🏛️ Vínculo Institucional e de Pesquisa
Este software é parte integrante das investigações acadêmicas desenvolvidas no âmbito do **GETMEP (Grupo de Estudos Teórico-Metodológicos em Educação e Pesquisa)**. O grupo dedica-se ao aprimoramento das práticas de pesquisa e à criação de tecnologias que qualifiquem a produção científica contemporânea.

🔗 **Visite nosso site:** [https://getmep-study.vercel.app/](https://getmep-study.vercel.app/)

---

## 🛠️ Funcionalidades
* **Interface Dinâmica:** Campos de entrada organizados por fases (Identificação, Triagem, Elegibilidade e Inclusão).
* **Exportação Profissional:** Suporte para salvamento em **PNG** (imagem de alta resolução para documentos) e **SVG** (vetor editável).
* **Design Oficial:** Estrutura visual idêntica ao modelo proposto pela diretriz atualizada.
* **Autossuficiência:** Software compilado para Windows, dispensando a instalação de ambientes como Python ou R.

## 📖 Referência Metodológica (Citação Obrigatória)
Ao utilizar diagramas gerados pelo FlowStudio em publicações acadêmicas (teses, dissertações ou artigos), é fundamental citar a diretriz base:

> **Page MJ, McKenzie JE, Bossuyt PM, Boutron I, Hoffmann TC, Mulrow CD, et al. A declaração PRISMA 2020: diretriz atualizada para relatar revisões sistemáticas. Epidemiol Serv Saude. 2022;31(2):e2022107.**
> **DOI: [10.5123/S1679-49742022000200033](https://doi.org/10.5123/S1679-49742022000200033)**

## 💻 Instalação
1. Vá para a seção de **Releases** deste repositório.
2. Baixe o arquivo `Instalador_FlowStudio.exe`.
3. Execute o assistente de instalação (disponível em Português do Brasil).
4. Utilize o atalho criado na Área de Trabalho para iniciar o software.
